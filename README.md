# Grant Chen's Portfolio

This is the source code for my personal portfolio website, built with React and Vite.

**Live page:** [https://grantchen08.github.io/grant-chen-portfolio/](https://grantchen08.github.io/grant-chen-portfolio/)

## About

This portfolio showcases my projects and skills as an aspiring multidisciplinary designer. It is designed to be a living document that reflects my journey as I learn and grow.

## Running Locally

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/grantchen08/grant-chen-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd grant-chen-portfolio
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:5173`.

## Deployment

This project is deployed to GitHub Pages using the `gh-pages` package. To deploy the website, run the following command:

```bash
npm run deploy
```
